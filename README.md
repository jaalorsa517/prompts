# 🚀 Prompts para Gema y Agentes Inteligentes

¡Bienvenido! Este repositorio es una colección curada y estratégica de prompts de alto rendimiento diseñados para potenciar la interacción con **Gema (Gemini)** y el ecosistema de agentes inteligentes.

Aquí no solo encontrarás instrucciones, sino arquitecturas completas para que la IA trabaje con precisión quirúrgica.

## 🧠 Lo que incluye este Proyecto

### 🤖 Agente Arquitecto de Prompts (v1.0)
Nuestra joya de la corona. En la sección `agent_sys_prompt`, encontrarás un agente especializado cuyo único propósito es **generar y optimizar prompts**.
- Capaz de construir **System Prompts** robustos para nuevos agentes.
- Diseña **User Prompts** optimizados para extraer el máximo potencial de cualquier LLM.
- Incluye un set de **Rules** y **Skills** modulares.

### ✨ Gema-Gemini
Estructuras diseñadas específicamente para el ecosistema de Google. Aprovecha las capacidades multimodales y de razonamiento avanzado de las Gemas de Gemini con frameworks optimizados.

## 🛠️ Estructura del Proyecto

- `/agents_sys_prompts/agent`: El núcleo del generador de prompts (System Prompt, Reglas y Habilidades).
- `/agents_sys_prompts/gema-gemini`: Configuraciones maestras para Gemas de Google.
- `/agents_sys_prompts/rules`: Librería de comportamientos estandarizados.
- `/agents_sys_prompts/skills`: Capacidades extendidas listas para integrar.
