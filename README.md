# Prompts para Gema y Agentes

Este repositorio contiene una colección de prompts especializados para interactuar con Gema (Gemini) y agentes inteligentes. El objetivo es proporcionar estructuras, reglas y habilidades para mejorar la eficiencia y precisión en el desarrollo y uso de modelos de lenguaje.
